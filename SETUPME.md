# 🚀 Project Setup (React + Vite + TypeScript)

This project is built using **Vite**, **React**, and **TypeScript** for a fast and modern development experience.

---

## 📦 Prerequisites

Make sure you have installed:

- Node.js (LTS version recommended) 👉 [https://nodejs.org](https://nodejs.org)
- npm (comes with Node.js)

Check installation:

```bash
node -v
npm -v
```

---

## ⚡ Create the Project

Run the following command:

```bash
npm create vite@latest my-react-app
```

---

## 🧩 Select Configuration

When prompted, choose:

- **Framework:** React
- **Variant:** TypeScript

---

## 📁 Move into the project folder

```bash
cd my-react-app
```

---

## 📦 Install dependencies

```bash
npm install
```

---

## ▶️ Start development server

```bash
npm run dev
```

Then open your browser at:

```
http://localhost:5173
```

---

## 🧱 Project Structure

```
src/
 ├── components/
 ├── App.tsx
 ├── main.tsx
index.html
package.json
tsconfig.json
```

---

## ✏️ Edit your first component

Open:

```
src/App.tsx
```

Example:

```tsx
function App() {
  return (
    <div>
      <h1>Hello React + Vite + TypeScript 🚀</h1>
    </div>
  );
}

export default App;
```

---

## 🚀 Build for production

```bash
npm run build
```

---

## 🧪 Preview production build

```bash
npm run preview
```

---

## 💡 Notes

- Vite provides instant hot module replacement (HMR)
- TypeScript improves code safety and scalability
- Ideal setup for modern React applications
